Parcial 1 - RauwAlejandroViews.
Santiago Santos Del Valle y Pablo Damian Lara Pérez.
Aplicación móvil desarrollada con .NET MAUI que muestra información relevante del artista Rauw Alejandro. El proyecto implementa navegación entre múltiples pantallas, incluyendo Inicio, biografía, discografía, giras pasadas y próximas presentaciones. Se aplican conceptos fundamentales de desarrollo multiplataforma como layouts (StackLayout, Grid), data binding básico, estilos globales mediante ResourceDictionary y organización modular de vistas. Además, el proyecto fue desarrollado de forma colaborativa utilizando control de versiones con GitHub.
1. Clonar el repositorio desde GitHub.
2. Abrir la solución en Visual Studio con soporte para .NET MAUI instalado.
3. Restaurar los paquetes NuGet del proyecto.
4. Seleccionar el dispositivo o emulador de destino.
5. Ejecutar la aplicación.
La app iniciará mostrando la navegación principal con la información del artista.
